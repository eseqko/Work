📄 Export to Google Doc

This generates a Google Apps Script with all 0 of your current courses embedded. Copy it, paste it at script.google.com, click Run — and a complete, formatted Course Catalog appears in your Google Drive in about 30 seconds.

No API keys. No setup. Runs entirely inside Google's own tools.

  1. 1Click Copy Script below — the entire script is copied to your clipboard.
  2. 2Click Open Apps Script ↗ → you'll land at script.google.com. Click New Project.
  3. 3Select all the starter code (Ctrl+A / ⌘A) and paste your copied script.
  4. 4In the toolbar, make sure the function dropdown (left of ▶) says myFunction. Then click ▶ Run. Google will ask for permissions — click Review Permissions → Allow.
  5. 5Wait ~2–5 minutes (many courses takes time). Then open Google Drive — your new doc will be there.
⚡ Google Apps Script  ·  paste into script.google.com/home/start → New Project
💡 After it finishes, go to Google Drive → My Drive and look for the new doc. You can also grab the direct link from Apps Script → Execution Log after the run completes. Each run creates a new doc, so only run it once per update.
